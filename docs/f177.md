 # Unit 16　省略句型
 ## Lesson 177　There be+主语+（省略that/who）从句
 
> 句型“Therebe+主语+（省略that/who）从句”中“that”和“who”作为从句的主语时可以省略。而省略的只能是“that”和“who”，不能是“which”。
